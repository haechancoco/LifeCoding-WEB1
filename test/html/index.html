Action 속성

action속성은 폼 데이터(form data)를 서버로 보낼 때 해당 데이터가 도착할 URL을 명시합니다. 

<form action="URL">

Target 속성

target속성은 폼 제출 후 수신된 응답을 표시할 위치를 지정합니다.

해당 target속성은 다음 값 중 하나를 가질 수 있습니다.

value Description
_blank 응답은 새 창이나 탭에 표시됩니다.
_self (기본값) 응답은 현재 창에 표시됩니다.
_parent 응답은 부모 프래임에 표시됩니다.
_top 응답은 맨 위 프래임에 표시됩니다.
framename 응답은 명명된 iframe에 표시됩니다.
