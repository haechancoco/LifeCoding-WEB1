Category (문서 유형)

  • Standard Track (표준 트랙): 공식적인 인터넷 표준 절차를 밟고 있는 문서입니다. (지켜야 할 규격)
  • Informational (정보성): 표준은 아니지만, 유용한 정보나 기술적인 조언을 제공합니다. (참고 자료)
  • Experimental (실험적): 연구 목적으로 제안된 기술입니다. 실제 서비스에 적용할 때는 주의가 필요합니다.
  • Best Current Practice (BCP): 표준은 아니지만, 운영상 "이렇게 하는 것이 가장 좋다"고 합의된 권장 사항입니다.
