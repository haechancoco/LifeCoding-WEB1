HTML TABLES

HTML Tables

table을 사용하면 데이터를 행과 열로 배열할 수 있습니다.

HTML table에 사용되는 태그

Tag Description
<table> 테이블을 정의합니다
<caption> 표의 목적을 명확하고 간결하게 설명합니다
<thead> Groups the header content in a table
<colgroup> Specifies a group of one or more columns in a table for formatting
<col> Specifies column properties for each column within a <colgroup> element
<th> Defines a header cell in a table
<tbody> Groups the body content in a table
<tr> Defines a row in a table
<td> Defines a cell in a table
<tfoot> Groups the footer content in a table

<caption>

<caption>은 표의 설명 또는 제목을 나타냅니다.

<caption>은 <table>의 첫 번째 자식이어야 합니다.

<colgroup>

<colgroup은 테이블의 열 집합을 정의합니다.

Attribute 설명 속성값 의미
span 연속적으로 얼마만큼의 열을 차지할 지를 결정합니다 0이 아닌 양의 정수 기본값: 1 속성값 만큼의 열을 차지함

Note: <colgroup>에 <col> 태그가 하나라도 있을 시 이 속성은 허용되지 않습니다.

<col>

<col>은 열 집합 내에서 1개 이상의 열을 정의합니다.

<col>은 <colgroup>을 부모로 두어야 합니다.

Attribute 설명 속성값 의미
<span> 연속적으로 얼마만큼의 열을 차지할 지를 결정합니다 0이 아닌 양의 정수 기본값: 1 속성값 만큼의 열을 차지함

Note: span 속성이 없는 <colgroup>에서만 작동합니다.

<th>

속성 설명 속성값 의미
<colspan 셀이 차지할 열의 개수를 정의합니다 음이 아닌 정수 기본값: 1 속성값 만큼의 열을 차지함
<rowspan> 셀이 차지할 행의 개수를 정의합니다 음이 아닌 정수 기본값: 1 속성값 만큼의 행을 차지함

<td>

속성 설명 속성값 의미
<colspan 셀이 차지할 열의 개수를 정의합니다 음이 아닌 정수 기본값: 1 속성값 만큼의 열을 차지함
<rowspan> 셀이 차지할 행의 개수를 정의합니다 음이 아닌 정수 기본값: 1 속성값 만큼의 행을 차지함
