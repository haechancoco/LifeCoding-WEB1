HOW TO MAKE PORTABLE CHROME

설치 파일 다운로드

구글 크롬 공식 사이트에서 .deb 파일을 다운로드합니다.

Bash 
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

파일 추출

dpkg 명령어를 사용하여 설치하지 않고 내용을 특정 폴더(chrome-portable)에 추출합니다.

Bash 
mkdir chrome-portable
dpkg -x google-chrome-stable_current_amd64.deb chrome-portable

실행

추출된 경로로 이동하여 실행합니다. 이때 중요한 것은 --user-data-dir 옵션을 사용하여 개인 정보(북마크, 쿠키 등)를 시스템 폴더(~/.config)가 아닌 현재 폴더 내에 저장하도록 설정하는 것입니다.

Bash 
# 실행 파일이 있는 경로로 이동
cd chrome-portable/opt/google/chrome

# 데이터 저장소를 현재 폴더의 'profile' 디렉토리로 지정하여 실행
./google-chrome --user-data-dir=./profile
